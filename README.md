### 현재 사용 가능 기능

#### 어땠어?

- 식당 별점 입력
- 메뉴 별점 입력
- 식당 정보 입력

#### 뭐 먹지?

- 식당 랜덤 추천
- 메뉴 랜덤 추천
- 식당 정보 표시(기본 정보, 별점 통계, 코멘트, 메뉴 정보, 메뉴 별점 통계, 메뉴 별점 코멘트)
- 식당 검색
- 메뉴 검색

### 구현 예정 기능

- 좀 더 다양한 랜덤 추천 옵션(e.g 가까운 거리, 별점 일정 기준 이상, ...)
